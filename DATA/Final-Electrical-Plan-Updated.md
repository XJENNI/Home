## INSTALLATION PHASES

### Phase 1: Pre-Construction (Before Plastering)
**Week 1-3 | Rough-In**

- Chisel grooves for concealed conduits (20/25/32mm PVC)
- Install junction boxes at ceiling-wall intersections
- Run conduits: DB → all switch/socket/light/fan points
- Mark recessed light positions on ceiling
- Pull cables with 300mm slack
- Label all conduit ends
- Test continuity with multimeter

### Phase 2: Post-Plastering (Finishing Work)
**Week 4-6 | Fixtures**

- Install metal mounting frames flush with plaster
- Connect switches/sockets to cables
- Cut precise holes for recessed lights
- Install LED drivers above false ceiling
- Mount chandelier (center, 8-10ft clearance)
- Hang BLDC fans with concrete anchors
- Install bathroom LED mirrors (touch sensor wiring)
- Install outdoor IP65 gate/facade lights

### Phase 3: Testing & Commissioning
**Week 7 | Safety & QC**

- Earth continuity test (<1 ohm)
- Insulation resistance test (>1 megaohm)
- RCCB trip test (30mA, <300ms)
- Polarity test (phase to switch)
- Test all switches control correct fixtures
- Verify 230V AC at all sockets
- Label MCBs in DB with permanent marker
- Document as-built drawings
